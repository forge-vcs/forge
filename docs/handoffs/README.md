# Handoffs

Session-handoff documents that let another agent — or a future you — pick up mid-task with full context.

Written by the `handoff` skill. Copy the skill's `mktemp` output here, then start the next session from the inline copy-paste prompt the skill prints.

## Convention

- Plan-linked handoff: `<plan-stem>-phase-<n>.md`
  (e.g. `2026-05-28-004-feat-competing-local-attempts-plan-phase-2.md`).
- Standalone handoff: `<YYYY-MM-DD>-<kebab-slug>.md`.

A good handoff captures: what's done, what's in flight, the next concrete step, and any open decisions — enough that the next session starts without re-deriving state.
